Agora, se tu quer manter a pasta vazia no repositório, cria o .gitkeep:

bash
Copiar código
mkdir -p arquivos
echo "" > arquivos/.gitkeep
git add arquivos/.gitkeep
git commit -m "Mantendo pasta arquivos vazia"
git push origin main
